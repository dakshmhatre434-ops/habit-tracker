# Habit Tracker Frontend

React + Vite + Tailwind CSS frontend for the Habit Tracker API.

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Axios
- React Router DOM
- Lucide React (icons)

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm run dev
```

Frontend runs at http://localhost:5173

Backend must be running at http://localhost:3000

## Features

- Dashboard with summary cards
- Habit cards with streak/completion stats
- Add habit modal with validation
- Weekly analytics modal with day-by-day breakdown
- Responsive sidebar layout
- Loading and empty states
- Error handling
